# BLU case study assets

This folder is the target for self-hosted BLU assets. The page currently
embeds Framer-hosted URLs directly (see the `<RemoteImage>` and
`<RemoteVideo>` components in `04-build/src/pages/blu/primitives.jsx`).
Each remote include carries a `TODO: download from Framer and self-host`
comment so they're easy to find.

## To self-host

Download each Framer asset listed below into this directory using the
suggested filename. Then in each section file, swap the `<RemoteImage src={...}>`
URL for `/blu/<filename>` and drop the TODO comment.
