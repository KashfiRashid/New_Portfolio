# BC Connect case study — asset drop zone

Drop the four assets below into this folder using the exact filenames.
Files here are served at `/bc-connect/FILENAME`. Each filename matches a
`<AssetPlaceholder>` slot in the case study — once a file is in place,
swap that placeholder for an `<img src="/bc-connect/FILENAME" .../>` (or
a `<video>` for a clip).

The same guidance renders inside each dashed placeholder on the live
page, so you can also just open `/work/bc-connect` and read the slots
in context.

---

## Slot 1 — Hero landing
- **File:** `hero-landing.png`
- **Where it shows:** top of the case study (Hero section)
- **Dimensions:** 2400 x 1500, 16:10, static screenshot
- **Content:** Full BC Connect landing page at desktop width. Hero
  headline, value proposition, search bar, and the lattice background
  all in frame. First image the reader sees, use the cleanest capture.
- **Annotate:** none.

## Slot 2 — Open Ground system
- **File:** `open-ground-system.png`
- **Where it shows:** end of The System section
- **Dimensions:** wide capture, static screenshot
- **Content:** The actual Open Ground artifact — the Figma component
  library page, the token sheet, or the style-guide document. This is
  the real proof behind the case study's whole argument.
- **Annotate:** optional — circle one token used across two different
  components to make the inheritance visible.

## Slot 3 — Integration composite
- **File:** `integration-composite.png`
- **Where it shows:** What I Did section, under ACTION 04
- **Dimensions:** 2400 x 1500, 16:10, static screenshot
- **Content:** Two or more shipped product surfaces side by side
  (directory next to dashboard, or landing next to a detail page),
  showing the same card primitive and spacing across both.
- **Annotate:** optional — a thin connector line between the same
  component as it appears on two surfaces.

## Slot 4 — Shipped directory
- **File:** `directory-shipped.png` (or `directory-shipped.mp4`)
- **Where it shows:** Results section, under the opening paragraph
- **Dimensions:** 2400 x 1500, 16:10, still or short clip
- **Content:** The directory page in use — card grid, FilterPills row,
  search bar, map toggle. A short screen recording of filtering in
  action is stronger than a still here.
- **Annotate:** none.
